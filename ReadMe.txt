This Project using React, Ruby on Rails and PostgreSQL

Follow Step to Install:

React:
- Install Node: https://nodejs.org/en
- Install React (vite.js): https://vitejs.dev/

Ruby on Rails:
- Install Ruby: https://www.ruby-lang.org/en/documentation/installation/

- Install Ruby On Rails (Framework): https://guides.rubyonrails.org/v5.1/getting_started.html

PostgreSQL:
- Install PostgreSQL 


*Follow Step to Run:*

1. Backend
Use terminal:
- Bundle install
- rails db:create
- rails db:migrate
- rails server

2. Frontend
- npm run dev
